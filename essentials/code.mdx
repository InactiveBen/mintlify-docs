## Overview

Fluent is a user interface library designed for Roblox, allowing developers to create customizable and interactive windows, tabs, and components with ease.

## Creating a Window

This is how you create the actual window for your interface.

```lua
local Main = require(game:GetService("ReplicatedStorage"):WaitForChild("Fluent"):WaitForChild("MainModule"))

local Window = Main:Window{
    Title = `Fluent {Main.Version}`, -- This is your title
    SubTitle = "by Actual Master Oogway", -- This is the sub title (light text next to title)
    TabWidth = 160, -- This is the tab width
    Size = UDim2.fromOffset(830, 525), -- This is the size of the interface
    Resize = true, -- Whether the window can be resized
    Acrylic = true, -- Acrylic on or off (it blurs anything behind the interface, requires graphic quality 8+)
    Theme = "Vynixu", -- The themes supported can be found in the themes category.
    MinimizeKey = Enum.KeyCode.RightControl -- Key to minimize or close the interface
}
```

## Creating Tabs

You can create multiple tabs within your window. Each tab can contain different UI elements.

```lua
local Tabs = {
    Main = Window:Tab{
        Title = "Main", -- The title of the tab
        Icon = "circle-user-round" -- The icon for the tab
    },
    Settings = Window:Tab{
        Title = "Settings", -- The title of the settings tab
        Icon = "settings" -- The icon for the settings tab
    }
}
```

## Adding a Paragraph

To add a paragraph to a tab, you can use the `Paragraph` method.

```lua
local Paragraph = Tabs.Main:Paragraph("Paragraph", {
    Title = "Paragraph",
    Content = "This is a paragraph.
Second line!"
})

print(Paragraph.Value)

Paragraph:SetValue("This paragraph text is changed!")

print(Paragraph.Value)
```

## Adding a Button

You can add buttons with callbacks for user interaction.

```lua
Tabs.Main:Button({
    Title = "Button",
    Description = "Very important button",
    Callback = function()
        local D = Window:Dialog({
            Title = "Title",
            Content = "This is a dialog",
            Buttons = {
                {
                    Title = "Confirm",
                    Callback = function()
                        Window:Dialog({
                            Title = "Another Dialog",
                            Content = "Lorem ipsum dolor sit amet, consectetur adipiscing elit. Suspendisse mollis dolor eget erat mattis, id mollis mauris cursus. Proin ornare sollicitudin odio, id posuere diam luctus id.",
                            Buttons = { { Title = "Ok", Callback = function() print("Ok") end} }
                        })
                    end
                },
                {
                    Title = "Cancel",
                    Callback = function()
                        print("Cancelled the dialog.")
                    end
                }
            }
        })
        print(D)
        D.Closed:Wait()
        print("Ok!!!")
    end
})
```

## Adding a Toggle

You can include toggle switches for binary options.

```lua
local Toggle = Tabs.Main:Toggle("Toggle", { Title = "Toggle", Default = false })

Toggle:OnChanged(function(Value)
    print("Toggle changed:", Value)
end)
```
